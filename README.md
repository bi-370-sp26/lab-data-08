# BI 370 — Lab 08 Data

This repository contains the raw CSV datasets for **Lab 08** of BI 370.

## Branches

| Branch | Description |
|--------|-------------|
| `main` | Shared baseline data for all sections |
| `section/01` – `section/06` | Per-section branch |

## Data Files

| File | Description |
|------|-------------|
| `data/cell_counts.csv` | Cell viability counts across species and treatments |
| `data/gene_expression.csv` | Fold-change and p-values for selected genes |
| `data/plant_growth.csv` | Shoot height and root length over time |
| `data/enzyme_kinetics.csv` | Reaction rates at varying substrate concentrations |
| `data/microscopy_measurements.csv` | Cell morphology measurements |
| `data/soil_microbiome.csv` | Relative abundance of soil bacterial phyla |
| `data/pcr_results.csv` | Ct values and efficiency from qPCR runs |
| `data/body_weight_timeseries.csv` | Mouse weight over 12 weeks by diet group |
| `data/water_quality.csv` | Water chemistry from field sampling stations |
| `data/hemocytometer_counts.csv` | Manual hemocytometer cell counts |

## Downloading Data

Click the green **Code** button and select **Download ZIP**, or clone with:

```bash
git clone https://github.com/bi-370-sp26/lab-data-08.git
```

To download only your section's branch:

```bash
git clone --branch section/01 https://github.com/bi-370-sp26/lab-data-08.git
```
